**Quiz Application**

A simple console-based Java application that simulates a multiple-choice quiz.
This mini project demonstrates how core Java concepts such as classes, objects, constructors, arrays, methods, loops, conditional statements, and user input handling can be used together to build a small interactive application.
The program asks users a series of questions, accepts their answers, checks correctness, and calculates the final score.

**Project Features**
The application allows users to:
Display multiple-choice quiz questions
Accept user answers through console input
Check whether the selected answer is correct
Calculate and display the final quiz score
Use objects to represent questions
Store questions in arrays and process them sequentially.
